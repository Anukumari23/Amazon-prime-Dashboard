# Amazon-prime-Dashboard
**Project Overview**

The Amizone Prime Movies Dashboard is an interactive Power BI project created to analyze and visualize movie data available on the Amizone Prime platform. The dashboard provides insights into movie genres, release years, ratings, duration, languages, and overall content distribution to help understand trends and user preferences.
**Short Description / Purpose**
This dashboard helps users explore and analyze movies available on Amizone Prime using dynamic and interactive visualizations. It converts raw movie data collected from Kaggle into meaningful insights for data analysis and portfolio demonstration.
